快速入门

1、什么是JAZZ？

JAZZ是一种基于jQuery的用户界面插件集合。为创建现代化，互动，JavaScript应用程序，提供必要的功能。 使用JAZZ你不需要写很多代码，你只需要通过编写一些简单HTML标记，就可以定义用户界面。节省网页开发的时间和规模。

2、引用文件

要引用jquery1.8，jquery ui，jazz-all.js

要引用jazz-all.css，在lib/themes 中有几个主题的样式

3、如何声明一个JAZZ组件

1. 直接在 HTML 声明组件。

<div name="test" id="_test" vtype="textfield" label="这是一个测试输入框" width="700" labelwidth="300"> </div> 2. 编写 JavaScript 代码来创建组件。 <div id="_test" > </div> $('#_test').textfield({ name: 'test', label: '这是一个测试输入框', width: '700', labelwidth: '300' });

组件有很多的属性，例如 vtype（组件名）、label、width、labelwidth等都是组件的属性，更多的属性请参考 API，组件可以生成后动态改变它的属性 $(selecter选择器).targetName组件名('option必须值','attribute属性名','arguement1参数','arguement2参数') 例如$('#_test').textfield('option','readonly',true)把textfield只读模式。

组件的方法，例如reset、reload、setValue、getValue等都是组件的方法，更多的方法请参考API， 组件的方法调用为 $(selecter选择器).targetName组件名('method方法名','arguement1参数','arguement2参数') 例如$('#_test').formpanel('setValue',data); 就是调用formpanel的setValue方法，传入的参数为data。

组件的事件，例如change、enter、itemselect等都是组件的事件，更多的事件请参考API， 组件的事件绑定为$(selecter选择器).targetName组件名("option", "event事件", function(event,ui){}); 或者$(selecter选择器).on("targetName组件名+event事件", function(event,ui){}) 例如$('#_test').textfield('option','enter',function(){alert('change);}) 给输入框绑定了一个失去焦点的事件，当它失去焦点时弹出 change。
