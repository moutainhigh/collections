边框设置，layoutconfig="{border: true}
拖动设置, layoutconfig="{border: true, draggable: true}

注意：布局内的vtype组件宽度width设置成“%”形式，布局被拖动后，vtype组件也会随之改变。
rowdragstart（拖动开始）和rowdragstop(拖动结束)是拖动列布局时的拖动事件。
参数一： e, 参数二： layout
layout.index 当前选中的边线的索引值
rowdragstop:
参数一： e, 参数二： layout
layout.top 拖动后，边线上部区域的宽度
layout.bottom 拖动后，边线下部区域的宽度
layout.index 当前选中的边线的索引值
