<static> jazz.tabpanel.addTab(taboption)

Parameters:
taboption
tab页url路径 { tabid: '', tab页的id tabtitle: '', tab页的标题 tabindex: '', tab的索引位置 taburl: '', tab页的url tabcontent: '' tab页的内容 }
taburl是链接到指定地址文件
tabcontent是展示具体内容
taburl和tabcontent这两个参数只需要定义一个即可, 优先使用taburl
