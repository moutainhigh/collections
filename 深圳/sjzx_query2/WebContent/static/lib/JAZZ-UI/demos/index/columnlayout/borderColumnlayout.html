边框设置，layoutconfig="{border: true}
拖动设置, layoutconfig="{border: true, draggable: true}

注意：布局内的vtype组件宽度width设置成“%”形式，布局被拖动后，vtype组件也会随之改变。
columndragstart（拖动开始）和columndragstop(拖动结束)是拖动列布局时的拖动事件。
参数一： e, 参数二： layout
layout.index 当前选中的边线的索引值
columndragstop:
参数一： e, 参数二： layout
layout.left 拖动后，边线左侧区域的宽度
layout.right 拖动后，边线右侧区域的宽度
layout.index 当前选中的边线的索引值
